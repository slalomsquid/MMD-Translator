# MMD-Translator
This addon translates the Japanese and Chinese words typically found in MMD models into English as well as fixing symbols and characters that may cause issues.

It is made as a replacement / fix to the inbuilt translatons in MMD-Tools as their list is not very exhaustive and tends to cause issues with leaving underscores.

# Installation
Drag the zip folder from the 'Releases' tab onto blender 4.0.0 or above
